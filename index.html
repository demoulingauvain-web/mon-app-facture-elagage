Guide d'Utilisation Enedis

1. Format du Fichier Excel

Structure Enedis attendue :

Mode HTA (R6) :
  • • Colonne 26 : Code Article
  • • Colonne 28 : Quantité réalisée (m)
  • • Colonne 34 : Tension (N = ST)
  • • Colonne 46 : Date
Mode BT (R4) :
  • • Colonne 22 : Code Article
  • • Colonne 24 : Quantité réalisée (m)
  • • Colonne 29 : Tension (N = ST)
  • • Colonne 50 : Date

2. Identification des Ouvrages

Ouvrage reconnu par format : 12345A1234

3. Archivage Automatique

Les factures sont classées par Prestataire / Mode / Année / Mois.
